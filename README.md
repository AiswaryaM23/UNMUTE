# 🗣️ Unmute

### AI-Powered Bidirectional Sign Language Communication System

Unmute is an AI-powered mobile application designed to facilitate seamless communication between deaf or speech-impaired individuals and the hearing community. The system integrates real-time Sign-to-Text translation, Text-to-Sign conversion, and conversational AI to provide an accessible and inclusive communication platform. By combining Computer Vision, Machine Learning, and Natural Language Processing, Unmute helps reduce communication barriers in educational, professional, and everyday environments.

---

## 🧠 AI & Machine Learning Concept

* MediaPipe is used for real-time hand landmark detection and extraction of 21 hand keypoints.
* A Random Forest classifier is trained to recognize American Sign Language (ASL) gestures.
* Natural Language Processing techniques are applied for sentence formation and text refinement.
* Gemini API enables intelligent chatbot interactions using text or recognized sign language input.

---

## ✨ Key Features

* Real-time Sign-to-Text translation using Machine Learning.
* Text-to-Sign conversion through ASL gesture image mapping.
* AI-powered chatbot for intelligent conversational support.
* Interactive ASL learning guide for users.
* Cross-platform Flutter application with an accessible user interface.
